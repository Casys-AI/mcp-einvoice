### M2 — `unescape(encodeURIComponent(...))` déprécié

- **Test** :
  `einvoice_invoice_generate_cii - handles non-ASCII characters (accents)`

### M3 — Auto-refresh absent dans `StatusTimeline` et `DoclistViewer`

- **Fichier** : `ui/status-timeline/`, `ui/doclist-viewer/`,
  `ui/directory-card/`
- **Fix** : Renommé `REFRESH_INTERVAL_MS` → `REFRESH_THROTTLE_MS` (nom reflète
  l'usage réel : throttle du refresh on-focus, pas timer auto). Pas de
  `setInterval` ajouté — pour des listes et timelines, le refresh
  on-focus/visibility est suffisant. Seul InvoiceViewer garde son timer (facture
  unique = statut peut changer en temps réel).
- [x] Corrigé

### M4 — `inv.way` vs `metadata.direction` inconsistance

- **Fichier** : `tools/invoice.ts:279`
- **Fix** : Ajouté fallback `inv.way ?? inv.metadata?.direction`. Les deux
  endpoints retournent la direction sous des clés différentes — le fallback
  couvre les deux cas.
- **Tests** : 2 tests — direction depuis `way`, fallback vers
  `metadata.direction`.
- [x] Corrigé

### M5 — `DirectoryCard` inutile pour `peppol_check`

- **Fichier** : `tools/directory.ts`, `ui/directory-card/`
- **Fix** : Retiré `_meta.ui` de `peppol_check` (le résultat booléen n'a pas la
  shape attendue par DirectoryCard).
